# game_chess_JS

#### 介绍
JavaScript 写的的中国象棋程序

#### 软件架构
```
- css: 样式文件

- img: 象棋图标文件

- js: 运行逻辑文件 
- - AI.js: ai对象，查看棋谱并查找最佳算法
- - bill.js: 棋谱存储对象
- - common.js：com对象，静态资源布局
- - gambit.js：开局布置
- - play.js：执行逻辑对象
- - store.js

- index: html入口文件
```
#### 安装教程

浏览器直接打开index.html 文件即可

注意修改index.html文件的编码格式： gbk & utf-8
